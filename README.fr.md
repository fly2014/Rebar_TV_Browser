Rebar TV Browser est un outil de lecture multimédia conçu pour le divertissement à domicile. Il est compatible avec les téléviseurs anciens et récents ainsi que les box TV, et prend en charge divers systèmes NAS.

Il inclut une fonction de navigateur avec simulation de souris via télécommande. Les utilisateurs peuvent ajouter des sites web via un code QR sur mobile, sans souris externe.

Il dispose d’un moteur de navigation double cœur, prend en charge la navigation verticale et un rendu web en 1080P niveau ordinateur. Il fonctionne également sur les anciens téléviseurs Android.

Sans publicité 🍺. Il est adapté à la lecture prolongée de pages web et de vidéos sur TV. Il offre toutes les fonctionnalités d’un navigateur classique sur Android TV, box TV, projecteur, stick TV et tablette.

Fonction lecteur

Lecture multimédia WEBDAV | HTTP | EMBY | NAS | M3U | USB (WebDAV). Chargement automatique des sous-titres. Décodage HD tous formats.
