鋼筋電視瀏覽器 是一款家庭娛樂多媒體播放工具，適配各類新舊電視、機上盒，並支援各種 NAS 存取。

具備瀏覽器功能，支援遙控器模擬滑鼠操作，可透過手機掃碼添加網站，無需外接滑鼠即可操作。

採用雙核心瀏覽引擎，支援直向瀏覽與 1080P 電腦級網頁渲染，適用於老舊 Android 電視。

無廣告 🍺。適合長時間在電視播放網頁與影片。支援 Android TV、機上盒、投影機、電視棒、平板等裝置。

播放器功能

WEBDAV | HTTP | EMBY | NAS | M3U | USB（WebDAV）。自動字幕載入。全格式高清解碼。
