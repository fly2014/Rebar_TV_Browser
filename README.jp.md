🇯🇵 日本語

Rebar TV Browser（鋼筋テレビブラウザ）は、家庭用エンターテインメント向けのマルチメディア再生ツールです。新旧さまざまなテレビ、テレビボックスに対応し、各種NASへのアクセスをサポートします。

ブラウザ機能を備えており、リモコンによるマウス操作のシミュレーションが可能です。スマートフォンのQRコードスキャンでサイトを追加でき、外部マウスなしで操作できます。

デュアルコアブラウザエンジンを搭載し、縦画面ブラウジングおよび1080PのPCレベルのWebレンダリングをサポートしています。古いAndroidテレビでも使用可能です。

広告なし🍺。テレビ上でWebページや動画を長時間再生するのに適しています。通常のブラウザと同様の機能をすべて備えており、Android TV、テレビボックス、プロジェクター、TVスティック、タブレットに幅広く対応します。

🎬 プレイヤー機能

マルチメディア再生機能として、WEBDAV | HTTP | EMBY | NAS | M3U | USB ネットワーククラウドドライブ（WebDAV）をサポート。字幕の自動読み込み、全フォーマットのHDデコードに対応し、さまざまなテレビ、工業用PC、テレビボックスで使用できます。

📦 インストール方法

APKをダウンロードし、USBに入れてテレビに挿してインストールしてください。

スマートフォンでQRコードをスキャンしてサイトを追加（テレビと同じWiFiルーターに接続している必要があります）。

QRが使えない場合：アプリをインストール後、テレビをWiFiに接続し、一度電源を抜いて再起動してください。
