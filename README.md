Rebar TV Browser / 钢筋电视浏览器（多语言版）
🇨🇳 中文

Rebar TV Browser 钢筋电视浏览器是一款面向家庭娱乐与工业场景的多媒体浏览工具，适配各类电视、电视盒子、投影仪与工控设备。

支持老旧 Android 设备（Android 5+），提供接近桌面级的网页渲染能力，并针对电视操作进行深度优化。

✨ 核心功能
支持双内核浏览引擎
1080P 高清网页渲染
电视遥控器 / 模拟鼠标操作
手机扫码快速添加网站（需同一 WiFi）
NAS / WebDAV / HTTP / EMBY / M3U / USB 播放支持
自动字幕加载
全格式高清视频解码
支持竖屏网页浏览
无广告体验
🏭 工业场景支持
MES 系统展示
ERP / 数据驾驶舱
工厂车间屏幕
商场 / 门店广告屏
生产报表展示
