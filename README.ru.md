Rebar TV Browser — это мультимедийный инструмент для домашнего развлечения. Поддерживает старые и новые телевизоры, ТВ-боксы и доступ к различным NAS системам.

Имеет функцию браузера с имитацией мыши через пульт. Можно добавлять сайты через QR-код с телефона без внешней мыши.

Использует двухъядерный движок браузера, поддерживает вертикальный просмотр и рендеринг уровня ПК в 1080P. Работает на старых Android TV.

Без рекламы 🍺. Подходит для длительного воспроизведения веб-страниц и видео. Поддерживает Android TV, ТВ-боксы, проекторы, стики и планшеты.

Функции плеера

WEBDAV | HTTP | EMBY | NAS | M3U | USB (WebDAV). Автосубтитры. Поддержка всех форматов HD.
