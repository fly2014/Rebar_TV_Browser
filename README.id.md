Rebar TV Browser adalah alat pemutar multimedia yang dirancang untuk hiburan rumah. Mendukung TV lama dan baru, TV box, serta akses ke berbagai NAS.

Memiliki fungsi browser dengan simulasi mouse menggunakan remote. Pengguna dapat menambahkan website melalui QR code di ponsel tanpa mouse eksternal.

Menggunakan mesin browser dual-core, mendukung browsing vertikal dan rendering 1080P seperti PC. Dapat berjalan di Android TV lama.

Tanpa iklan 🍺. Cocok untuk pemutaran jangka panjang halaman web dan video di TV. Mendukung semua fungsi browser untuk Android TV, TV box, proyektor, TV stick, dan tablet.

Fitur Player

Mendukung WEBDAV | HTTP | EMBY | NAS | M3U | USB (WebDAV). Subtitle otomatis. Dekoding video HD semua format.
