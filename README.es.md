Rebar TV Browser es una herramienta de reproducción multimedia diseñada para entretenimiento en el hogar. Es compatible con televisores antiguos y modernos, TV boxes y soporta acceso a varios sistemas NAS.

Incluye función de navegador con simulación de ratón mediante control remoto. Permite añadir sitios web mediante código QR desde el móvil, sin necesidad de ratón externo.

Cuenta con un motor de doble núcleo, soporta navegación vertical y renderizado web en 1080P nivel PC. Funciona en televisores Android antiguos.

Sin anuncios 🍺. Es ideal para reproducción continua de páginas web y videos en TV. Ofrece todas las funciones de un navegador estándar en Android TV, TV box, proyectores, sticks y tablets.

Funciones del reproductor

Soporta WEBDAV | HTTP | EMBY | NAS | M3U | USB (WebDAV). Subtítulos automáticos. Decodificación HD completa.
